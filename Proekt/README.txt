Для запуска (в консоли):
    (опционально)
        1. python -m venv .venv
        2. - Windows
                .\.venv\Scripts\activate
           - Unix
               source .\.venv/bin/activate
    (обязательно)
        1. (один раз) pip install -r requirements.txt
        2. python main.py
        3. После заполнения ответов, в папке с файлом 'main.py' появится файл result.png
